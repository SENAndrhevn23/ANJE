# ANJR

ANJRE (Andre Nicholas Jr Engine) is a fork of Psych Engine 1.0.4 with better optimizations and extra features.

Sadly, Psych Engine was archived on March 31, 2025. This engine is supposed to continue what Psych Engine started, while adding more features, better performance, and improved support for large projects.

---

# Features

* Rendering Modes
* Engine Optimizations
* Can load over 15MB JSON charts
* Spammy Feature in Chart Editor (similar to JS Engine)
* More features coming soon

---

# Planned Features

* Mobile support
* More rendering improvements
* Better chart editor tools
* Additional optimization updates

# Goals

This engine is planned to be worked on very actively and aims to become a strong alternative to other major Psych Engine forks like H-SLICE.

It may take years to fully reach that goal, but development will continue over time.
